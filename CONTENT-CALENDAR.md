## Publishing Schedule Template

### Blog Post Checklist
- [ ] Target keyword identified (primary + 2-3 secondary)
- [ ] Word count meets minimum (1,200+ for guides, 800+ for profiles)
- [ ] Meta title (50-60 chars)
- [ ] Meta description (150-160 chars)
- [ ] H1 optimized
- [ ] Internal links (3-5 to relevant products/categories)
- [ ] External links (1-2 to authoritative sources)
- [ ] Images (3+ with optimized alt text)
- [ ] Schema markup (Article or BlogPosting)
- [ ] FAQ section where applicable
- [ ] CTA to shop/browse
- [ ] Social sharing images created

### Product Description Checklist
- [ ] Unique content (not duplicated)
- [ ] 400+ words
- [ ] Primary keyword in first 100 words
- [ ] Weaver/cooperative attribution
- [ ] Origin story included
- [ ] Specifications table
- [ ] Styling suggestions
- [ ] Care instructions
- [ ] Product schema implemented
- [ ] 5+ high-quality images
